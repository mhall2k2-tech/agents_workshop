# Sleep-Memory 2x2 Analysis Summary

Date: 2026-03-06

## Design
- Between-participants 2 x 2 factorial design.
- Factors: Sleep (Sleep vs Wake) and Cue (TMR vs Control).
- Dependent variable: recall_score (0-40).
- Total N = 80 (n = 20 per cell).

## Cell descriptives
- Sleep-Control: M = 26.49, SD = 3.58
- Sleep-TMR: M = 29.85, SD = 3.92
- Wake-Control: M = 21.58, SD = 3.36
- Wake-TMR: M = 24.54, SD = 3.22

## Factorial ANOVA
- Sleep main effect: F(1, 76) = 42.01, p < .001, partial eta squared = 0.356
- Cue main effect: F(1, 76) = 16.01, p < .001, partial eta squared = 0.174
- Sleep x Cue interaction: F(1, 76) = 0.07, p = .798, partial eta squared = 0.001

## Interpretation (data-bound)
- Sleep condition was associated with higher recall scores.
- TMR cueing was associated with higher recall scores.
- No evidence of a Sleep x Cue interaction was observed in this dataset.
